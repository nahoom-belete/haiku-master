# Haiku Master

A short survival game where you collect keys while avoiding enemies. Gather every key to unlock a personalized haiku at the end of the game.

## Features

* Survival-focused gameplay
* Key collection objective
* Enemy avoidance mechanics
* Personalized haiku reward
* Playable in the browser

## Installation

1. Clone the repository.
2. Open the project in Unity.
3. Open the main scene.
4. Press Play in the Unity Editor.
